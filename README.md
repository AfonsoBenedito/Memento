# Memento - Site OMQ

Este é um projeto de álbum de fotografias desenvolvido para a disciplina de IC. É uma aplicação web totalmente do lado do cliente (client-side) que simula um sistema de gestão de fotografias.

## Estrutura do Projeto

O projeto foi otimizado e segue a seguinte estrutura:

- `index.html`: Página inicial (Login/Registo).
- `assets/`: Pasta contendo todos os recursos estáticos.
  - `css/`: Folhas de estilo.
  - `js/`: Scripts JavaScript.
  - `img/`: Imagens do sistema e fotografias de exemplo.
  - `fonts/`: Tipografia.

## Como Utilizar

1. **Abrir a Aplicação**:
   - Navegue até à pasta deste projeto.
   - Encontre o ficheiro `index.html`.
   - Abra-o com o seu navegador de internet preferido.

2. **Registo**:
   - O sistema utiliza `LocalStorage`.
   - Preencha o formulário de "Registar" com dados à sua escolha.

3. **Login**:
   - Utilize os dados criadaos para entrar.

## Notas Técnicas

- **Persistência**: Dados guardados no `LocalStorage`.
- **Tecnologias**: HTML5, CSS3, JavaScript (Vanilla).
